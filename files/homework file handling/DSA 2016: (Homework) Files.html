(Homework) Files

* Write a program which concatenates one file at the end of another

 Notes: the program will be run as

./program  file1  file2

This will append the file "file1" at the end of the file "file2"

*  Write a program which reads a file and counts the no. of occurrences of every word in it.

* Write a program which reads a CSV file and prints n'th column.

Notes:

The program will be run as

./program -5  myfile.csv

This will print the 5th column from the file.

You have to think about reading a line (using a readline() function) and cutting it in "tokens" separated on "," using strtok() function.

* Write a program which removes every occurrence of a given word from a file.
Notes:
The program will be run as
./program  xyz   file1
This will remove all the occurrences of the word "xyz" from the file named as "file1"
A word is a char sequence separated by any whitespace (space, tab, newline).
Note that the "file1" will all have remaining content (including all whitespace) unaltered.
You may use other files temporarily.
