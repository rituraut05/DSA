11:07 AM:

Sample Problem Statement for the exam (for regular students, not for diploma students)

Problem:

First write an implementation of a list of doubles using doubly linked NULL terminated structures. Write the following functions:

void init(list *l);

void append(list *l, double x);

double remove(list *l, int pos); /* remove from a position */

void reverse((list *l);

void store(list *l, char *fname);  /* Stores the list in the file specified by fname */

Now, write an implementation of a stack of doubles, using array as storage. Write following functions:

void sinit(stack *s);

void push(stack *s, double x);

double pop(stack *s);

int sempty(stack *s);

int sfull(stack *s);


You can write other functions also for the above data types, if needed.

Now write a main function which does the following:

i) Takes two filenames as  arguments.

ii) Reads data from a text file (argv[1]) as given below, and carries out the said operations on a list.


a 10.2

i  11.1  0

i  12.21 1

i  -100.11  1

r  1


Where

a 10.2 --> means append 10.2 to the list

i 11.1 0 --> means insert 11.1 at position 0

r  1 --> means remove from position 1


After this the main function

iii) reverses the list.

iv) removes elements from location 0 continuously till list is empty, and pushes every removed element on the stack

v) pops all elements from stack and appends to a new list , till stack is empty

vi)  stores the list in a texxt file (specified as fname) using the store() function. This fname should be argv[2]
