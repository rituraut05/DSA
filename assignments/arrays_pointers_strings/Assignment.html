(Assignment) Strings and Maths Problems

Problem 1) [0.5 mark]

Write a program to compute x^y based on using base-3 presentation of a number. In the program, write a function which computs x^y

Filename: rollno_branch_batch_xpowy3.c

Problem 2) [0.5 marks]

Write a program to remove duplicate doubles from an array of doubles, without changing the ordering of the numbers.

In the program, write a function which accepts an array of doubles and removes the duplicates from the array and has return type void. 

Filename;  rollno_branch_batch_remdupl.c

Problem 3) [1 marks]

Write a complete C program, which has following function:

int stringreplace(char *text, char *orig, char *new);

This function replaces all occurances  of the string orig inside the string text to the string new and then returns the number of instances replaced. Note: the string text changes as a result of the program. Submit a single .c file having the name like  1110002011_comp_s2_stringreplace.c

Examples

1)

text:  thisismynamewhatishisname

orig:    his

new:   her

After function call, text becomes: therismynamewhatishername and function returns 2 as it did 2 replacements.

2)

text:  aabrakaadaabraaisgaabra

orig:   abr

new:  bing

after function call text becomes: abingakaadabingaaisgabinga and function returns 3 as it did three replacements.

Use the following main function:

#include <stdio.h>
int main() {
    char text[128], orig[128], new[128]; // don't assume size of 128 in the function stringreplace, it is not available in the function!
    int count;
    while(scanf("%s%s%s", text, orig, new) != -1) {
       count = stringreplace(text, orig, new);
       printf("%d %s\n", count, text);
    }
}

Filename: rollno_branch_batch_stringreplace.c

Submission: Submit both files in a .tar.gz file named as rollno_branch_batch_stringmaths.tar.gz
